## gstack

本项目已安装 `gstack` 到 `.claude/skills/gstack`。

- 使用 `gstack` 的 `/browse` 进行所有网页浏览相关工作。
- 不要使用 `mcp__claude-in-chrome__*` 工具进行浏览器操作。
- 可用命令（slash commands）：
  - `/plan-ceo-review`
  - `/plan-eng-review`
  - `/plan-design-review`
  - `/design-consultation`
  - `/review`
  - `/ship`
  - `/browse`
  - `/qa`
  - `/qa-only`
  - `/qa-design-review`
  - `/setup-browser-cookies`
  - `/retro`
  - `/document-release`

如果命令不可用或安装不完整，请在项目中重新执行：

```bash
cd .claude/skills/gstack && ./setup
```
